1. Does TRACE need ground truth trajectories for GlobalTargetPosGuidance?
   → NO ❌ 
   
   The model GENERATES trajectories guided by your waypoints. You only need:
   • Current agent state (automatically from scene)
   • Target waypoint position (you provide)
   • Guidance parameters (you set)

2. Can you synthesize trajectories by providing sparse waypoints?
   → YES ✅
   
   Example:
   waypoint = (10.5, 0.0)  # Where agents should go
   urgency = 0.8           # How fast (0=slow, 1=fast)
   → Model generates smooth trajectory reaching waypoint

3. Does the number of inferred frames depend on ORCA dataset?
   → NO ❌
   
   Inference length is controlled by config:
   cfg.num_simulation_steps = 100  # This controls frame count
   Same for all datasets (ORCA, nuScenes, Arena)

4. How to use with Arena world?
   → See WAYPOINT_GUIDANCE_GUIDE.md section 4 for detailed steps

VALID GUIDANCE CONFIG EXAMPLE:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

# Make 3 agents go to different waypoints with different speeds
guidance_config = {
    'name': 'global_target_pos',       # Guidance type
    'weight': 1.0,                      # REQUIRED! Guidance strength (1.0 = default)
    'params': {
        'target_pos': [                # Where each agent should go
            [10.5, 0.0],   # Agent 0
            [10.5, 2.0],   # Agent 1
            [8.0, 1.0],    # Agent 2
        ],
        'urgency': [0.8, 0.8, 0.6],    # Speed: 0=slow, 1=fast
        'pref_speed': 1.42,             # Expected walking speed (m/s)
        'dt': 0.1,                      # Timestep (always 0.1 for TRACE)
        'min_progress_dist': 0.5,       # Min progress per step (m)
    },
    'agents': [0, 1, 2]  # Which agents to apply to
}

# CRITICAL: Wrap in [[config]] not [config]
# Structure: [[config for scene 0], [config for scene 1], ...]
guidance_config_list = [[guidance_config]]  # List of scenes, each with list of configs

# Use in inference:
results = guided_rollout(
    env=env,
    policy=rollout_policy,
    policy_model=policy_model,
    guidance_config=guidance_config_list,
    horizon=100,
)
